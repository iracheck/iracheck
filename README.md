## My Projects

### Embedded
- **[Checkers Robot](https://github.com/iracheck/checkers-bot)**  
  A physical robot capable of playing checkers against humans or AI. Combines a Python game engine with a C-controlled robotic arm. My role in the project was the programming side of the brain & robotic arm controls, while two mechanical engineers and an electrical engineer worked on other aspects of the project.<br>
  **Technologies:** Python, C, OpenCV, PySerial  
  **Timeline:** Feb 2026 – Present  
  **Status:** In Progress

### Modding / Simulation
- **[Dynamic Squad AI Simulation](https://github.com/iracheck/DynamicSimulationMod)** (*GangWarSandbox*)  
  Developed a high-performance, extensible combat simulation framework with modular game modes, lifecycle hooks, and custom faction/vehicle configs via `.ini`. Over **200 downloads** as of Jan 2026.  
  **Technologies:** C#/.NET, ScriptHookV, LemonUI  
  **Timeline:** V1.0: June 2025 – Aug 2025, V1.1: Jan 2026 <br>
  **Status:** Completed; limited support/occasional updates

### Full-Stack / Accessibility
- **[PCRemote](https://github.com/iracheck/pcremote-csharp)**  
  A desktop utility for Windows 11 that translates game controller input into precise mouse and keyboard actions. Ideal for accessibility and non-native game integration.  
  **Technologies:** C#/.NET, WPF/XAML, XInput  
  **Timeline:** March 2025  
  **Status:** Project shuttered (baseline version fully functional)
